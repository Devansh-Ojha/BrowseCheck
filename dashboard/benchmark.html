About the benchmark

Lakera PINT Benchmark: A neutral evaluation of prompt injection detection systems. Not trained on the PINT dataset to avoid overfitting.

BrowseCheck approach: Uses Claude with a targeted system prompt to detect prompt injections. Designed for agent browser security.

Score calculation: Balanced accuracy = mean of accuracy on positive and negative examples (avoids bias toward the more common class).
